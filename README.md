# Script de configuração inicial para distros Ubuntu Based!
Esse é meu script atual para o ano de 2026 para distros ubuntu based como Pop_!OS, Linux Mint e o proprio Ubuntu. O script foi desenvolvido para sanar as minhas necessidades como usuario de distros Ubuntu-based! para qualquer modificação fique avontade para realizar um `fork` deste repositório e implementar ou decrementar o que quiser!

### Funcionalidades
- Configuração de distro ubuntu-based com base na minha experiência.
- Adição de repositórios Oficiais e Não-Oficiais.
- Instalação de pacotes basicos.
- Instalação e configuração de ambiente de desenvolvimento.
